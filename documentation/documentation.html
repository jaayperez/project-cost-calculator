Overview

Costy is an ultra responsive, multipurpose cost calculator and order wizard template. It is based on Bootstrap4 so looks great on any device.
With the help of Costy you can provide your customers or visitors a platform where they can easily calculate prices of any of your products or services.
Besides that they will be able to generate and send an order list of the choosen products to you and themselves with few clicks.

Now Costy comes with 3 demo versions and with 8 color variations: demo-1, demo-2 and demo-3 has the same structure (as listed below):

  • blue
  • coral
  • green
  • light-blue
  • light-green
  • orange
  • purple
  • yellow

HTML Structure

The template is built up using HTML5, CSS3 and jQuery.
The layout is centered, using Bootsrap framework grid so looks great on any device.
It contains eye catching web parts like: icons and form elements (toggles, input fields, checkboxes).
Has great user experience with: preloader, input effects, popup images, animations and user notifications.
Works with smooth scroll navigation.

CSS Structure

The core stylesheet files are in css folder

  • main...css - styles of them main pages (index.html, faq.html)
  • order...css - styles of the calculator and order wizard pages: order-1(-2-3).html)
  • custom.css - your custom style can go here

The main and the order css files have the following premade color skin variations:

  • main.blue.css, order-1.blue.css, order-2.blue.css, order-3.blue.css
  • main.coral.css, order-1.coral.css, order-2.coral.css, order-3.coral.css
  • main.green.css, order-1.green.css, order-2.green.css, order-3.green.css
  • main.light.blue.css, order-1.light.blue.css, order-2.light.blue.css, order-3.light.blue.css
  • main.light.green.css, order-1.light.green.css, order-2.light.green.css, order-3.light.green.css
  • main.orange.css, order-1.orange.css, order-2.orange.css, order-3.orange.css
  • main.purple.css, order-1.purple.css, order-2.purple.css, order-3.purple.css
  • main.yellow.css, order-1.yellow.css, order-2.yellow.css, order-3.yellow.css

The vendor stylesheet files are placed in vendor folder

  • bootstrap
  • font-awesome
  • icomoon (icon fonts)
  • lightbox
  • parsley
  • pure-css-checkbox

The minified stylesheet files are in css\min folder

The compressed styles are minified and beggining with: styles keyword (e.g.: styles.blue.min or styles.order-1.purple.min.css).
These compressed styles are containing every necessary core and vendor styles in 1 file. This is very useful from peformance, SEO (Search Optimization) point of view.
If you uncomment (activate) and use the detailed CSS please do not forget to comment out (deactivate) the compressed part in order to avoid conflicts.

Color Change

Open Video about changing skin on new tab

This color replacement is needed for all html and php files if you would like to setup a predefined color skin.

If you are working with compressed version of styles then it is enough just pick another compressed style with the desired color.
(E.g.: if the default template comes with blue skin and if you would like to change it to yellow then you have to change styles.blue.min.css to styles.yellow.min.css in the index.html
and change styles.order-1.blue.min.css to styles.order-1.yellow.min.css in order-1.html).

PHP Code Explanation

PHP code can be setup in order_send.php.
Please only change your email (2. row), your subject (3.), "from" item (4.) subject for the customer (20.), "from email address" (21.) and confirmation message (24.) 

$mail = $_POST['email'];
$to = "websolutions.ultimate@gmail.com"; // Your email goes here
$subject = "Order from Costy";
$headers = "From: Costy ";
$message = "SUMMARY\n";
$message .= "\nFirst name: " . $_POST['firstname'];
$message .= "\nLast name: " . $_POST['lastname'];
$message .= "\nEmail: " . $_POST['email'];
$message .= "\nPhone: " . $_POST['telephone'] . "\n";
$message .= "\nORDER\n";
$message .= "--\n" . $_POST['order'];
$message .= "\nAdditional message: " . $_POST['message'];
$message .= "\nTerms and conditions accepted: " . $_POST['terms'];

//Receive Variable
$sentOk = mail($to,$subject,$message,$headers);

//Confirmation Page
$user = "$mail";
$usersubject = "Thank You for your order";
$userheaders = "From: info@yourdomain.com\n";

//Confirmation Page With Summary
$usermessage = "Your order is successfully submitted. Thank you. We will get back to you shortly.\n\n$message";
mail($user,$usersubject,$usermessage,$userheaders);

JavaScript

The core script files are in js folder

  • main.js - functionality for main pages
  • order-1.js - calculator and order wizard scripts
  • order-2.js - calculator and order wizard scripts
  • order-3.js - calculator and order wizard scripts

The vendor script files are placed in vendor folder

  • bootstrap
  • easing
  • jquery
  • lightbox2
  • parsley
  • scrollreveal

If you need, you can change your currency in the beginning of order-1(-2-3).js

The minified stylesheet files are in js\min folder

The compressed scripts are minified and beggining with: scripts keyword (e.g.: scripts.min.js or scripts.order-1.min.js).
These compressed scripts are containing every necessary core and vendor scripts in 1 file. This is very useful from peformance, SEO (Search Optimization) point of view.
